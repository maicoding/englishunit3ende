# English Unit 3/4 Trainer

Kleines statisches Lernpaket fuer iPad/Safari.

## Inhalt

- `index.html` - Vokabeltrainer mit Spaced Repetition, Spelling, Papier-Check, Testmodus und Textanalyse-Training
- `README_Mutter_1_Seite.md` - kurze Installations- und Nutzungsanleitung
- `druckmaterial/` - Druckboegen fuer Papiertraining, Fehlerliste, Tests und Textanalyse-Raster

## Nutzung

Die Seite kann direkt auf Netlify deployed werden. Danach den Link auf dem iPad in Safari oeffnen und ueber `Teilen -> Zum Home-Bildschirm` speichern.

Der Lernstand wird lokal im Browser des jeweiligen iPads gespeichert. Es werden keine Lernstaende an einen Server gesendet.
